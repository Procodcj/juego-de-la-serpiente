# 🐍 Juego de la Serpiente

Un clásico juego de la serpiente desarrollado en Python con Pygame, con estética retro-oscura.

## Requisitos

- Python 3.7 o superior
- Pygame

## Instalación

1. Clona el repositorio:
```bash
   git clone https://github.com/Procodcj/juego-de-la-serpiente.git
   cd juego-de-la-serpiente
```

2. Instala Pygame:
```bash
   pip install pygame
```

## Cómo jugar
```bash
python snake.py
```

## Controles

| Tecla | Acción |
|-------|--------|
| ↑ ↓ ← → | Mover la serpiente |
| Enter | Iniciar / reiniciar |
| Esc | Salir |

## Reglas

- Come la comida roja para crecer y sumar puntos (+10 por comida)
- Si chocas contigo misma, el juego termina
- La serpiente puede atravesar los bordes y aparecer al lado opuesto

## Tecnologías

- Python
- Pygame
